Accessibility Statement

Our commitment to making this website accessible to everyone.

Our Commitment

Faubion Choir is committed to ensuring that our website is accessible to people with disabilities. We strive to meet or exceed the requirements of the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA, which is the standard referenced by the Americans with Disabilities Act (ADA).

Accessibility Features

This website includes the following accessibility features:

  • Semantic HTML — Proper heading hierarchy, landmarks, and page structure for screen readers
  • Keyboard Navigation — All interactive elements are fully accessible via keyboard
  • Skip Navigation — A skip link allows keyboard users to jump directly to main content
  • Color Contrast — All text meets WCAG AA minimum contrast ratios (4.5:1 for normal text, 3:1 for large text)
  • Responsive Design — The site works on all screen sizes and supports text resizing up to 200%
  • Focus Indicators — Visible focus outlines on all interactive elements
  • ARIA Labels — Descriptive labels for navigation, buttons, and interactive elements
  • Alternative Text — Descriptive alt text for all meaningful images
  • Form Accessibility — All form fields have associated labels and required field indicators
  • Reduced Motion — Animations are disabled when users prefer reduced motion
  • High Contrast Support — Enhanced styles for users who prefer high contrast
  • Link Context — External links indicate when they open in a new window

Feedback

We welcome feedback about the accessibility of this website. If you encounter any barriers or have suggestions for improvement, please contact us. We take accessibility concerns seriously and will work to address any issues promptly.

Standards

This website aims to conform to:

  • WCAG 2.1 Level AA
  • Section 508 of the Rehabilitation Act
  • ADA Title II (as applicable to public school programs)
